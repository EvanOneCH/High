# High一下

High一下，点击 http://zhengxiaopeng.com 里的 `High一下` 试用。   
修复点击重复执行播放的问题，重新点击后动画重新开始，加了几首歌点击随机播放。

## Usage

把如下超链接放到网站上需显示的位置，请注意修改 `high.js` 里的配置信息。

``` Html
<a href='high.js里的代码复制进来'>High一下</a> 
```
